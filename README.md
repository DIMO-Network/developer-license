# DIMO Developer License

This is an NFT collection minted for developers building on the DIMO Network.

## Deployment Script

Using `TestToken` instead of `$DIMO`.

```
npx ts-node ./script/test_deploy.ts
```

Using `$DIMO`. 

```
npx ts-node ./script/prod_deploy.ts
```
